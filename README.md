## 關於這個專案

**技術棧**
- React
- JSX
- React DnD
- SCSS

**特殊的元素**
- 具RWD功能，判別使用者裝置呈現相對應版面配置及顏色。如為裝置為電腦時計算機則可拖曳，使用套件`React DnD`達成此功能；如為裝置為手機時計算機的高度為畫面50%，超出的範圍則可使用scroll功能(使用`overflow`)。
